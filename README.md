## Project Overview
Implementation of custom data structures without using `java.util.*` classes.

## Components
1. **MyList (Interface)**: Defined standard operations for all list types.
2. **MyArrayList**: Array-based list with dynamic resizing.
3. **MyLinkedList**: Doubly linked list for efficient head/tail manipulation.
4. **MyStack**: LIFO structure built on `MyArrayList`.
5. **MyQueue**: FIFO structure built on `MyLinkedList`.
6. **MyMinHeap**: Min-priority queue using array-based binary tree logic.

## Key Learnings
- Managing generic types with `Comparable`.
- Time complexity differences ($O(1)$ vs $O(n)$) between arrays and linked lists.
